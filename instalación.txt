Para la instalacion correcta de la base de datos correspondiente a 'emprendimiento_condimento' primero debes ingresar a la carpeta archivosSQL y

luego debes ejecutar los siguientes pasos/archivos en el siguiente orden:

1- create_database_&_tables: En este archivo se creara la base de datos y las tablas correspondientes.

2- data_insert: Este archivo contiene la inserciones para las tablas previamente creadas.

3- script_views: En este archivo podemos ver la creacion de 5 vistas de la base de datos.

4- functions: En este archivo se crearan las 2 funciones que corresponden a la base de datos.

5- stored_procedure: En este archivo se crearan los 2 procedimientos.

6- script_triggers: En este archivo podemos observar la creacion de las dos tablas de auditoria y los 4 triggers para la base de datos.

7- DCL: En este archivo podemos ver la creacion de los usuarios y darles los permisos a cada uno con relacion a nuestra base de datos.

8- TCL: En este archivo podemos visualizar, iniciando una transaccion, una sentencia de modificacion hacia algunas de las tablas.
